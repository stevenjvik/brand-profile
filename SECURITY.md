# Security Policy

## Reporting a vulnerability

Please email **security@stevenjvik.tech** with a reproducer and affected area. Expect acknowledgement within 72 hours.

Do **not** open a public issue for security bugs.

## Scope

This repo is the GitHub profile README. There is no executable code. Reportable issues are limited to:

- Misleading content that could be used for impersonation or social engineering.
- Links that redirect to malicious destinations.

## Supported versions

The `main` branch is the only version. There are no releases.
